# PR Learning Note Template

## 목적

PR packet을 읽은 뒤 해석과 학습을 같은 형식으로 남기기 위한 템플릿이다.

이 문서는 `playbooks/pr-learning-pipeline.md`의 해석/학습 단계를 기록할 때 기본 형식으로 사용한다.

---

## 템플릿

```md
# <미션명 / 주제명>

## 1. 질문

- 이번에 확인하려는 질문

## 2. 데이터 범위

- 대상 미션:
- 사용한 DB:
- 사용한 packet:
- 대표로 다시 본 PR:

## 3. 관찰

- 반복해서 보인 패턴
- 예외적으로 달랐던 패턴
- 인상적이었던 리뷰 코멘트

## 4. 해석

- 리뷰어가 실제로 문제 삼은 지점
- 그 지점이 왜 문제였는지
- 내가 지금 이해한 판단 기준

## 5. 내 코드와 연결

- 내 코드에서 비슷한 구조
- 그대로 가져갈 점
- 바꿔야 할 점

## 6. 다음 질문

- 아직 남아 있는 질문 1
- 아직 남아 있는 질문 2
```

---

## 최소 작성 기준

시간이 없으면 아래 네 줄만 남긴다.

- 무엇을 봤는가
- 무엇이 반복됐는가
- 내 코드와 어떻게 연결되는가
- 다음 질문이 무엇인가
